# Computer Workshop

مجموعه تمرین‌ها و پروژه‌های درس **کارگاه کامپیوتر**.

## درباره پروژه

این مخزن شامل تمرین‌ها و فعالیت‌های انجام‌شده در درس کارگاه کامپیوتر در طول ترم است.

## بخش‌های پروژه

### 📄 Word
تمرین‌های مربوط به نرم‌افزار Microsoft Word شامل:
- قالب‌بندی متن
- ایجاد جداول
- استفاده از تصاویر و اشکال
- تنظیم صفحه و استایل‌ها
- تهیه گزارش‌های متنی

### 📊 Excel
تمرین‌های مربوط به Microsoft Excel شامل:
- ورود و مدیریت داده‌ها
- فرمول‌ها و توابع
- رسم نمودارها
- تحلیل داده‌های ساده
- قالب‌بندی صفحات گسترده

### 📽️ PowerPoint
تمرین‌های مربوط به Microsoft PowerPoint شامل:
- طراحی اسلاید
- استفاده از تصاویر و نمودارها
- انیمیشن‌ها و انتقال‌ها
- آماده‌سازی ارائه‌های آموزشی

### 📝 Markdown
تمرین‌های مربوط به Markdown شامل:
- ساختاردهی اسناد
- تیترها و فهرست‌ها
- لینک‌ها و تصاویر
- جداول
- نگارش فایل‌های README

- Navigation Bar (منو ناوبری)
